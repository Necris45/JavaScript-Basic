7. *Сравнить null и 0. Попробуйте объяснить результат.
True будет возвращать только сравнения вида null >= 0 и null <= 0
Если найденная информация верна, то для нестрогого равенства == значений undefined и null 
действует особое правило: эти значения ни к чему не приводятся, они равны друг другу и не 
равны ничему другому.

При этом в некоторых статьях упоминается, что сравнение >= (<=) работает так:
если сравнение < (>) возвращает False, то >= (<=) автоматически принимается за True
